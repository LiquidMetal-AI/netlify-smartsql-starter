Seed Database

Initialize your SmartSQL database with sample sales data. This creates tables and inserts sample customers, products, orders, and order items.

Note: The seed button is idempotent - it's safe to click multiple times. If tables already exist, seeding will be skipped.

Database Schema

customers

  • id (INTEGER)
  • name (TEXT)
  • email (TEXT)
  • created_at (TEXT)

products

  • id (INTEGER)
  • name (TEXT)
  • price (REAL)
  • category (TEXT)

orders

  • id (INTEGER)
  • customer_id (INTEGER)
  • order_date (TEXT)
  • total_amount (REAL)

order_items

  • id (INTEGER)
  • order_id (INTEGER)
  • product_id (INTEGER)
  • quantity (INTEGER)
  • price (REAL)
